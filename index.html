First Single Page React Application: Feedback Form

About

This was my full fledged Single Page Application with routing and full page renders depending on the route. It was designed as a feedback form to provide instructors feedback on the day's lesson/lessons. It was also my first use of Redux as a way to manage global state. As the user goes through the app an object is created in redux, which is then send to the server and subsequently to the database on the submission confirmation.

You can find the repository at https://github.com/Samuelvphipps/weekend-redux-feedback-loop

Lessons learned

  • Semantic elements are just divs with names for accessibility and organization.
  • I really enjoy this and have a ton to learn.

First Full-Stack App: Grip 'N Grin

About

Grip 'N Grin was my first ever full stack application developed from scratch, including scoping (scoping document), programming, debugging and deploying (deployed website link.) I used React for the front end and Node.js for the backend.

You can view the project at: https://gripngrin.herokuapp.com/#/home.

Lessons learned

  • Always manage state and ensure that there is no state mismatch.
  • Stick to the single responsibility principle.
  • Comment clearly as you go.
  • Reasses development plan as the project develops.
